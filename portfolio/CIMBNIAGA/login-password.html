    Lupa password?

Isi password dengan keyboard Anda atau virtual keyboard (?) Mengenai Virtual Keyboard

Virtual keyboard adalah gambar keyboard QWERTY yang diproyeksikan ke monitor komputer Anda. Untuk menggunakannya, cukup arahkan dan klik mouse Anda ke gambar tombol.

Untuk keamanan, Anda disarankan untuk menggunakan virtual keyboard untuk memasukkan password untuk mengurangi resiko keystroke logging. Bagi malware, akan lebih sulit untuk mendapatkan data melalui keyboard virtual, daripada untuk memantau keystroke nyata.
